ShulNET 2.0 User Documentation

Welcome to the ShulNET 2.0 user documentation. This guide will help you understand and use all features of your synagogue management system.

New to ShulNET? Start with the Getting Started guide to learn the basics.

What is ShulNET?

ShulNET 2.0 is a comprehensive synagogue management system designed to help religious organizations manage their members, events, finances, and more. Built with modern technology, it provides an intuitive interface for both administrators and members.

System Requirements

ShulNET 2.0 works on all modern web browsers including:

  • Google Chrome (recommended)
  • Mozilla Firefox
  • Microsoft Edge
  • Safari (macOS and iOS)
Note: Internet Explorer is not supported. Please use a modern browser for the best experience.
